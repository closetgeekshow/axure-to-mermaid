Mobile App with

Sliding Nav


This page uses a dynamic panel to set up a scrollable area (Scroll Vertical) for the main content.


The sliding nav is a Hidden dynamic panel (yellow mask). It is shown when the menu icon is clicked.


There are sample buttons in the nav panel that change the content with the Set Panel State action and hide the nav.


The Loaded event on the nav panel moves it to 0,0 and brings it to the front so feel free to move it out of the way.
